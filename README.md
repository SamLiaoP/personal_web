# Sam Liao 個人網站

這是Sam Liao的個人作品集網站，展示了個人專業背景、專案和技能。

## 功能特色

- 響應式設計
- 動態滾動動畫
- 項目卡片互動效果
- 打字機效果標題
- 聯絡方式整合

## 技術堆疊

- HTML5
- CSS3
- JavaScript (vanilla)
- 字體圖標 (Font Awesome)
- Google Fonts

## 部署於Zeabur

本網站已部署於Zeabur平台，可直接訪問：
[https://sam-liao.zeabur.app](https://sam-liao.zeabur.app)

## 本地運行

1. 克隆此儲存庫
2. 使用任意HTTP伺服器啟動網站，例如：
   ```
   python3 -m http.server 8000
   ```
3. 瀏覽器訪問 http://localhost:8000
